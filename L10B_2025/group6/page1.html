Cloud Robotics Engineer

Cloud Robotics Engineer — це фахівець, який інтегрує роботизовані системи з хмарними сервісами. Його робота охоплює як робототехніку, так і хмарні обчислення, що дає змогу підвищити гнучкість, масштабованість та інтелектуальність роботизованих систем.

До обов’язків входить створення інфраструктури для зберігання та обробки даних від роботів у хмарі, реалізація віддаленого управління, використання AI/ML-моделей для прийняття рішень у режимі реального часу.

Використовуються такі інструменти: ROS, Docker, Kubernetes, AWS RoboMaker, Google Cloud Robotics, MQTT, REST API.
